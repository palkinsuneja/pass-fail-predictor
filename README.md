# 🎓 Pass/Fail Predictor

A beginner-friendly Machine Learning project that predicts whether a student will **Pass or Fail** based on study hours using **Logistic Regression**.

---

## 🚀 Demo

This app takes input (study hours) and predicts:
- ✅ Pass / ❌ Fail
- 📊 Probability of passing

---

## 🧠 How it Works

- We trained a Logistic Regression model on sample data
- The model learns the relationship between **study hours → result**
- A **sigmoid function** maps values to probability (0 to 1)
- If probability ≥ 0.5 → Pass  
- Else → Fail  

---

## 🛠 Tech Stack

- Python 🐍  
- NumPy  
- Streamlit (for UI)
